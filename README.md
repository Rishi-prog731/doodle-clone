## Run Locally

Clone the project

```bash
git clone https://github.com/Rishi-prog731/scribble-leap.git
```

Go to the project directory

```bash
cd scribble-leap
```

Install dependencies

```bash
npm install -g live-server
```

Start the server

```bash
http-server
live-server
```
